# Momento Espírita Alexa Skill Assets

This repository hosts public static image assets used by the unofficial Alexa skill **Mensagem Espírita de Hoje**.

The files in this repository are intended to be publicly accessible through GitHub Pages so they can be used as artwork or background images in Alexa `AudioPlayer` metadata.

This repository does not contain the Alexa skill source code, credentials, deployment logic, or private configuration.
